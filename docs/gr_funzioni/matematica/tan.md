# tan

Restituisce la tangente di un angolo.

## Sintassi

tan(_<span style="color:red;">angle</span>_)

## Argomenti

* _<span style="color:red;">angle</span>_ angolo in radianti

## Esempi

* `tan(1.0) → 1.5574077246549`
* `tan(pi()/2) → infinito`
