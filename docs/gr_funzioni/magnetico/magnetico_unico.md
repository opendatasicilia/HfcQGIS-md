# Gruppo Magnetico

!!! Abstract
    **Questo gruppo contiene funzioni per lavorare con i modelli di campo magnetico terrestre.**

---

## magnetic_declination

Restituisce la declinazione del campo magnetico in un punto, in gradi a est del nord.

Sintassi:

- magnetic_declination(_<span style="color:red;">point</span>_[,_<span style="color:red;">datetime</span>_])

[ ] indica componenti opzionali

Argomenti:

- _<span style="color:red;">point</span>_ geometria punto
- _<span style="color:red;">datetime</span>_ data e ora di riferimento per il calcolo (opzionale)

Esempi:

```
…
```

Feature introdotta a partire da **QGIS 4.0**
